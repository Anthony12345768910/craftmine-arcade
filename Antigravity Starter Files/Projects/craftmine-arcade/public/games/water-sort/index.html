Water Sort

Sort the potions. One color per tube. Don't spill!

How to Play

  • 🧪 Click a tube to pick up its top color
  • 🧪 Click another tube to pour it in
  • ✅ You can pour onto any color or into an empty tube
  • 🏆 Sort all tubes so each has only one color
  • ↩️ Use Undo if you make a mistake
  • 🔄 Use Restart to retry the level

# Skills — Reusable Patterns & Playbook

This is your agent's growing library of reusable skills, patterns, and workflows.
Every time you build something that could be used again — save it here.
Over time, this becomes the most valuable folder in your workspace.


# How to Use This Folder

Before a project:
- Read the index below and check what skills are available.

During a project:
- Agents can read and apply skill files instead of starting from scratch.

After a project:
- Save anything reusable as a new `.md` file and add it to the index.

# Skills Index

Add a new row every time you create a skill file.

- `auth-implementation-patterns/SKILL.md` — Safe patterns for login, signup, sessions, and user data.

# Skill File Template
When creating a new skill, copy this format into a new .md file:

```markdown
# Skill Name

> One sentence describing what this skill does.

---

## When to Use It
- [FILL — describe the situation where this applies]
- [FILL]

## Steps / Pattern
1. [FILL]
2. [FILL]
3. [FILL]

## Example
[FILL — paste a short code snippet or example]

## Notes & Gotchas
- [FILL — anything important to watch out for]
- [FILL]
```
